# 集米玩法落地盘（统一可复用轻量方案）

> 本篇定义**一套 12 城通用的城市站标准方案**：玩家做什么、现场长什么样、要哪些物料、怎么把假米交接到深圳。
> **不分大小档**——各城落地量级由执行同学按当地实际快速调整，方案侧只给"可复用模板 + 标准动作"。
> 状态：评审中（v0.1）

---

## 一、城市站定位（一句话）

**城市站 = 轻触点。** 玩家来这里只做一件事的体感：**完成三个小动作，领一份纪念礼包，给一座城添上自己的一粒米，顺手拍点能发的素材。**

不办大活动、不设复杂关卡、不绑跨城收集——**轻、快、能出片、能进群**。

---

## 二、玩家动线（4 步，全程 ≤ 5 分钟）

```
①到场 → ②做三件事 → ③往Logo箱添米 → ④领礼包+拍照打卡 → 离场
```

| 步骤 | 玩家做什么 | 现场承接 | 产出 |
|---|---|---|---|
| ① 到场签到 | 扫码/报到，了解玩法 | 引导台 + 一句话玩法牌 | 到场数据 |
| ② 做三件事 | 1.加分部群 2.投城市菜 3.留言墙留言 | 群活码 / 投票板 / 留言墙 | 进群、投票、UGC留言 |
| ③ 添米 | 把假米投进城市 Logo 箱 | Logo造型箱（缩小版） | "我给本城添了一粒米"的参与感 |
| ④ 领礼包+打卡 | 领【假米+成员证+普发周边】，拍照 | 礼包发放台 + 主视觉拍照位 | 晒图素材、成员证传播 |

> 设计原则：**三件事 = 进场即可顺手完成**，不排长队、不设淘汰、不卡门槛。

---

## 三、三件事细则（核心串联）

### 3.1 加分部群
- 现场**群活码**（按城市分群），扫码即入本城分部群
- 作用：把一次性到场玩家**沉淀为可长期触达的社群资产**（本次活动最重要的转化）
- 话术轻量：「进群解锁本城后续福利 + 村晚消息第一时间知道」

### 3.2 城市菜现场投票
- 每城预设 N 道**候选城市菜**（地方特色菜，玩家投票选出"本城送往深圳村晚的代表菜"）
- 形式：现场投票板（贴纸/盖章/扫码皆可，执行同学定）
- 钩子：**你投的菜，村晚真的会上桌**——把城市站和深圳终点提前咬合，制造"我参与的结果会在终点出现"的期待

### 3.3 留言墙
- 现场一面**留言墙**，玩家写下对米饭联盟/对这座城/对村晚的一句话
- 作用：**解决"个体不可见"**——让每个普通玩家的参与被看见、可被拍
- 延伸：城市站留言墙内容**汇总到深圳村晚现场展示**（线上也可做"全国留言墙"H5/长图，传播篇细化）

## 四、聚米装置（城市站版）

| 项目 | 说明 |
|---|---|
| 形态 | **米饭联盟 Logo 造型箱 · 缩小版**（透明/半透明，能看见米堆积） |
| 内容物 | **统一 3D 打印假米**（不强制玩家自带、不用真米，省运输/卫生/损耗） |
| 城市色 | 每城对应一个**城市代表色**；城市站可用色卡标识，最终深圳统一染色倒入灯箱（也可城市站即用色米，执行定） |
| 核心要求 | **箱子本身要好看、能出片、能当拍照主体**——它就是城市站的主视觉 |
| 数据感（可选） | 箱体旁可放一块小牌/小屏："本城已添米 XXX 粒 / 已有 XXX 位米饭参与"，把参与变成可见反馈 |

> 装置的意义不在"装多少米"，而在**它是城市站最强的那张照片**，也是"全国一路聚米"叙事的实体锚点。

## 五、物证三件套（普发礼包落地细则）

完成三件事即领，人人有份：

| # | 物证 | 落地要点 |
|---|---|---|
| ① | **一粒米**（3D打印假米） | 玩家投进Logo箱的同款，可带走1粒作纪念；批量备料 |
| ② | **米饭联盟·XX分部 成员证** | **纪念性周边**：正面分部名+城市色+主视觉，背面可留"饭搭子签名位/打卡位"。**非功能证件、不绑权益**，措辞不用"会员/认证/权益" |
| ③ | **一款普发周边** | 执行同学按库存/成本从备选品类选定：吧唧·徽章 / 贴纸 / 滴胶挂件·钥匙扣 / 镭射票·小卡 / 布章 等 |

**分部长专属款**（限量，单独设计）：
- 比普发成员证更特别一点的**留念物**（带编号/特殊工艺），亮明"我是这座城的组织者"的身份小荣誉
- 同为纪念性质，**不绑权益**

> 🔓 留口：若后续出现"好玩且轻量"的体验型彩蛋（如村晚现场凭证优先合影位），可作为成员证隐藏玩法绑定；默认不绑。

## 六、城市站现场分区（可复用模板）

一个城市站建议含 5 个功能点位（面积大小执行同学按场地缩放）：

| 点位 | 功能 | 出片价值 |
|---|---|---|
| 1. 引导签到台 | 报到、玩法说明、群活码 | — |
| 2. 城市菜投票区 | 投票板 | 中 |
| 3. 留言墙 | 玩家留言 | 高（人多时墙面满，画面强） |
| 4. Logo 添米装置区 | 投米、数据牌 | 高（主视觉） |
| 5. 礼包+拍照打卡位 | 领礼包、主视觉合影 | 高（成员证晒图源头） |

## 七、城市 → 深圳 假米交接流程

```
各城收束 → 假米按城市色归集 → 统一运抵深圳 → 倒入巨型Logo灯箱对应色区 → 村晚点亮
```

| 环节 | 要点 |
|---|---|
| 城市收束 | 活动结束后，城市站假米归集打包（标注城市+色） |
| 归集运输 | 统一物流到深圳村晚现场（执行排期，含时间冗余） |
| 现场入箱 | 各城色米倒入灯箱对应色区/色层，形成"全国拼成一个Logo"的画面 |
| 留言/投票同步 | 各城留言墙精选 + 城市菜投票结果，同步带到深圳现场展示/上桌 |

> 替代方案（执行可选）：城市站不实际运米，深圳统一备料按城市色填充，城市站假米仅作现场道具——**降低物流成本，画面效果一致**。两种都行，执行按成本定。
