02

集米玩法落地盘

CITY STATION · 统一可复用轻量方案

玩家动线（全程 ≤ 5 分钟）

① 到场签到→ ② 做三件事→ ③ 端电饭煲讨饭集米→ ④ 围炉吃饭盒 + 领礼包

不排队、不淘汰、不卡门槛，进场顺手完成。

核心串联：做三件事 → 讨饭集米 → 领礼包 + 米饭纪念票

👥
加分部群
把一次性到场玩家沉淀为可长期触达的社群——本次最重要的转化
🍲
城市菜投票
钩子：你投的菜，村晚真的会上桌，提前咬合深圳终点
✍️
留言墙留言
解决「个体不可见」，城市留言汇总到深圳现场展示

完成三件事 + 参与讨饭集米 → 领普发礼包，并获得一张 「米饭纪念票」（深圳吃席抽签时享更高中签权重，非直接资格，最终以官方报名抽签为准）。

普发礼包 · 3 件套（人人完成即得）

🌾
一粒米
玩家自带，或现场领 3D 打印假米；可带走 1 粒作纪念
🎫
米饭纪念票
纪念性周边 · 可秀可晒有梗。附带深圳吃席抽签加权，非直接资格
🎁
一款普发周边
执行按库存/成本选：吧唧 / 贴纸 / 滴胶 / 镭射票 / 布章 等
米饭纪念票口径：纪念周边 + 深圳吃席抽签更高中签权重，不等于直接资格，持票仍需抽签、不必然到场。对外文案统一按此，防「有票就能吃」误解。

核心道具与装置（填充物统一用假米）

  • 填充统一用大颗粒米道具（大气球米 / 棉花抱枕米），不用真米，省制作、易复用、控成本
  • 城市站 · 装置电饭煲（大）：放大版电饭煲，展示 + 互动打卡点，本城「米桶」主视觉
  • 城市站 · 道具电饭煲（小，可用真电饭煲）：玩家端着讨饭集米、拍内容
  • 深圳终点 · 仪式装置：巨型 Logo 灯箱透明装置，各城假米汇入填满 → 亮灯 → 开席

巡城玩法升级：讨饭集米 + 围炉吃饭盒

  • 讨饭集米：玩家端小电饭煲在现场及周边商圈「化缘讨米」，找 NPC / 玩家 / 路人讨一勺假米，凑够一煲倒进本城大电饭煲
  • 围炉吃饭盒：集米完成后围坐大电饭煲前一起吃饭盒，主打「蹲一块儿真吃饭」的团建感
仪式 = 围炉吃饭本身，不做念米经/添米礼/暗号等额外包装。

03

深圳终点站 · 吃饭大会

FINALE · 游乐园自由逛 + 漫展团建饭局

整体调性

  • 参照「游乐园自由逛 + 漫展团建饭局」，官方主导动作集中在开场亮灯、荣誉颁奖、全员合影，其余交给现场自然流动
  • 大部分时间留给玩家自由吃逛拍社交，降低参与压力，像一场朋友聚会而非演出
  • 城市菜揭晓 / coser 上菜 / 米饭整活三块尽量 玩家共创（PUGC），官方搭台、玩家唱戏
  • coser = 玩家也是 NPC：从报名玩家中挑选，担任跑堂/掌勺/任务发布，营造游乐园 NPC 的沉浸感
  • 到场米饭提前报名 → 筛选 → 核销入场，兼顾成本、人数与现场活跃度

公关 / 自媒体稿 · 只做收尾总结

不铺前置通稿，收尾集中发一篇总结，让行业与路人都看懂全貌。

  • 讲清一件事：一场「全国聚米 → 深圳开席」，把 12 城玩家组织成可见的大事
  • 给得出画面与数据：讨饭路线图、现场、coser 传菜、地域菜，配真实城市/到场/留言数据
  • 留有人味的故事：穿插一两个玩家/分部长参与故事

合规提示（广告法红线）

  • 全程规避极限词，不用「最/第一/国家级/唯一/顶级」等
  • 数据只用真实可核的数字（到场/留言/城市数），不编造、不夸大转化
  • 玩家信息收集（报名/进群）遵循个人信息保护要求，明确用途
  • 物证措辞守住：米饭纪念票是纪念周边，不暗示官方认证或特权
  • 米饭纪念票措辞守住：它是纪念票 + 抽签更高权重，不承诺「持票必到场」，吃席名单以官方报名抽签与名额为准，规则提前公示，防误解舆情
